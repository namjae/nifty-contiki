Function Details

compile/3

compile(InterfaceFile::string(), Module::module(), Options::options()) -> ok | fail

Generates a NIF module out of a C header file and compiles it, generating wrapper functions for all functions present in the header file. InterfaceFile specifies the header file. Module specifies the module name of the translated NIF. Options specifies the compile options. These options are equivalent to rebar's config options.

render/4

render(InterfaceFile::string(), ModuleName::modulename(), CFlags::[string()], Options::options()) -> {error, reason()} | {renderout(), [nonempty_string()]}

Renders an InterfaceFile into a Erlang module containing of ModuleName.erl ModuleName.c, ModuleName.app and rebar.config and returns the contents of these files as tuple of iolists (in this order). It uses CFlags to parse the InterfaceFile and Options to compile it. Options are equivalent to rebar options.
